## Version EN

I shipped 12 dev tools today.

Not as a product — as a pack. One command installs a menu, you pick what you need:
- a tool that reads an install script before you run it
- a tool that shares a package cache across a slow local network
- a tool that syncs a git repo via USB, no internet required

I'm based in Ouagadougou and I develop with AI pairing — slow connections and infrastructure gaps are a real constraint here, not an edge case. These tools are what I kept wishing existed.

Everything is open source: github.com/limack0/limack-devtools

Install: curl -fsSL get.limackcorp.online | sh

Built with Claude, pushed by me.

## Version FR

J'ai shipé 12 outils dev aujourd'hui.

Pas un produit unique — un pack. Une commande installe un menu, tu choisis ce dont tu as besoin :
- un outil qui lit un script d'installation avant que tu l'exécutes
- un outil qui partage un cache de paquets sur un réseau local lent
- un outil qui synchronise un dépôt git par clé USB, sans connexion

Je suis basé à Ouagadougou, je développe en binôme avec l'IA — les connexions lentes et les coupures d'infrastructure sont une contrainte réelle ici, pas un cas limite. Ces outils, c'est ce que je cherchais à chaque fois sans trouver.

Tout est open source : github.com/limack0/limack-devtools

Installation : curl -fsSL get.limackcorp.online | sh

Co-écrit avec Claude, structuré et relu par moi.
