# Cross-link snippet — paste into other repos' READMEs

A discreet footer that points back to the pack. Pick one.

## Badge

```markdown
[![Lim@ck DevTools](https://img.shields.io/badge/part_of-Lim%40ck_DevTools-1f6feb)](https://github.com/limack0/limack-devtools)
```

Renders as: a small blue **part of · Lim@ck DevTools** badge linking to the repo.

## One-line footer

```markdown
---
🧰 Built by the maker of [**Lim@ck DevTools**](https://github.com/limack0/limack-devtools) — offline-first dev tools, one command each: `curl -fsSL get.limackcorp.online | sh`
```

## Compact line (for tight READMEs)

```markdown
> 🧰 Part of [Lim@ck DevTools](https://github.com/limack0/limack-devtools).
```

Where to put it: at the very bottom of the README, after the license, so it never competes with the project's own content.
